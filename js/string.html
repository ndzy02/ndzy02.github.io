# charAt charCodeAt 不改变原字符串

charAt 返回在指定位置的字符;指定位置不存在返回 "" 空字符串

charCodeAt 返回在指定位置的字符的 Unicode 编码;指定位置不存在返回 NAN

const str = 'abc';
console.log(str.charAt(0)); // a
console.log(str.charCodeAt(0)); // 97

# concat 不改变原字符串

连接字符串 返回连接后的字符串

const a = 'abc';

console.log(c); // abcdef

# indexOf lastIndexOf 不改变原字符串

检索字符串 对大小写敏感 返回对应的索引;indexOf 从开始位置开始查找，lastIndexOf 从结束位置开始查找; 未找到返回-1

const str = 'Hello world!';
console.log(str.indexOf('Hello')); // 0
console.log(str.indexOf('World')); // -1
console.log(str.indexOf('world')); // 6

console.log(str.indexOf('o')); // 4
console.log(str.lastIndexOf('o')); // 7

# match 不改变原数组

在字符串中检索指定的值,或找到一个或多个正则表达式的匹配。返回指定的值 没有匹配值返回 null

const str = '1 abc 2 def 3123';
console.log(str.match(/\d+/g)); // ['1', '2', '3123'];

console.log(str.match('12')); // ["12", index: 13, input: "1 abc 2 def 3123", groups: undefined]

# slice 不改变原字符串

语法: str.slice(beginIndex[, endIndex])

  • str.slice(beginIndex,endIndex) 不包括 endIndex 所在的元素
  • 返回提取出来的新字符串; 否则返回 "" 空字符串
const str = 'abcdef';

// beginIndex 起索引，如果值为负数按照 strLength + beginIndex 处理
// endIndex 始索引，如果值为负数按照 strLength + endIndex 处理
str.slice(4, 5); // e
str.slice(2, -1); // cde 等价于 str.slice(2, -1 + 6)
str.slice(2); // cdef 等价于 str.slice(-4 + 6)

//str.slice(beginIndex) 从 beginIndex 索引所在元素开始至 str 结束
str.slice(1); // bcdef 等价于 str.slice(-5 + 6)
